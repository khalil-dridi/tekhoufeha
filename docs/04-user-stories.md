# User Stories

---

# Introduction

This document defines the user stories of the TekhouFeha platform from the perspective of its users. These user stories describe the expected behavior of the system and will serve as the foundation for the Product Backlog, Use Cases, and future development sprints.

---

# Visitor

## US-001 Browse Products

**As a Visitor**

I want to browse available products

So that I can discover products without creating an account.

---

## US-002 Search Products

**As a Visitor**

I want to search products using keywords

So that I can quickly find products that interest me.

---

## US-003 Filter Products

**As a Visitor**

I want to filter products by:

- Category
- Selling Mode
- Location
- Product Condition

So that I can easily find relevant products.

---

## US-004 View Product Details

**As a Visitor**

I want to view complete product information

So that I can decide whether I am interested.

---

## US-005 Create an Account

**As a Visitor**

I want to create an account

So that I can publish products and submit offers.

---

# User

## US-006 Login

**As a User**

I want to log into my account

So that I can access my personal dashboard.

---

## US-007 Publish a Product

**As a User**

I want to publish a second-hand product

So that other users can view it.

---

## US-008 Choose Selling Mode

**As a User**

I want to choose the selling mode

So that I can sell my product using either:

- Fixed Price
- Make an Offer

---

## US-009 Edit Product

**As a User**

I want to edit my published product

So that I can update its information.

---

## US-010 Delete Product

**As a User**

I want to remove my product

So that it is no longer visible.

---

## US-011 Submit an Offer

**As a User**

I want to submit an offer for a product

So that I can express my purchase interest.

---

## US-012 Cancel an Offer

**As a User**

I want to cancel my pending offer

So that I can submit a new one if necessary.

---

## US-013 View My Offers

**As a User**

I want to view all my submitted offers

So that I can track their status.

---

## US-014 Receive Offer Decision

**As a User**

I want to receive notifications when my offer is accepted or rejected

So that I know what happens next.

---

## US-015 Manage Received Offers

**As a User**

I want to review offers received on my products

So that I can decide which buyer to choose.

---

## US-016 Accept an Offer

**As a User**

I want to accept one offer

So that I can reserve my product for that buyer.

---

## US-017 Reject an Offer

**As a User**

I want to reject an offer

So that I can refuse unsuitable proposals.

---

## US-018 Keep an Offer Pending

**As a User**

I want to leave an offer pending

So that I can wait for additional offers before making my decision.

---

## US-019 Start a Private Chat

**As a User**

I want a private conversation to start only after an offer has been accepted

So that only serious buyers can contact me.

---

## US-020 Exchange Messages

**As a User**

I want to exchange messages after an offer has been accepted

So that we can organize the transaction.

---

## US-021 Receive Notifications

**As a User**

I want to receive notifications

So that I stay informed about platform activities.

---

## US-022 Manage Favorites

**As a User**

I want to save products to my favorites

So that I can easily find them later.

---

## US-023 View Dashboard

**As a User**

I want to access my dashboard

So that I can manage my products, offers, chats, and notifications.

---

# Administrator

## US-024 Manage Users

**As an Administrator**

I want to manage user accounts

So that the platform remains secure.

---

## US-025 Moderate Products

**As an Administrator**

I want to moderate published products

So that inappropriate content is removed.

---

## US-026 Review Reports

**As an Administrator**

I want to review user reports

So that I can take appropriate actions.

---

## US-027 View Platform Statistics

**As an Administrator**

I want to access platform statistics

So that I can monitor platform performance.

---

# Future User Stories

The following user stories are planned for future releases:

- AI price estimation.
- Product recommendation system.
- Reputation and rating system.
- Online payment.
- Delivery management.
- Mobile application.
- Premium subscriptions.
